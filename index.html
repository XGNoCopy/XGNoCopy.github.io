发表于 2018-04-11

前言

不知不觉，从事iOS开发4年了，连一个个人博客都没有，说出来你可能不信。但是我说我的Dota天梯1800，王者荣耀3个小号都是王者段位，吃鸡5000多分，你可能就信了，哈哈哈嗝。。。。为什么不写博客的我也要来写博客了呢？就是因为一时兴起而已。那这个博客是用来分享技术的吗？不不不。。。。技术只占了一小部分，目前iOS热门的技术，我想如果你随便Google一下或者百度一下，毫不夸张的说，相同的技术点都有成千上百的相同文章或者博客出现。程序员的大部分时间被技术占据，我们需要不断的丰富知识，总结技术历程，提升我们的技术能力这点毋庸置疑，但是闲暇时间我们就来玩点其他的。一件新的事物诞生了，我还是希望它有不一样的色彩，要不然就是一件山寨货了。

个人博客的诞生

关键词：Hexo + GitHub Pages + 个人域名绑定

由于GitHub Pages提供了展示个人静态网页的功能，所以这种方式构建的个人博客是不需要自己配置服务器空间的，完全免费，较为便捷。如果有需要可以考虑花钱买付费的服务器空间，但是一定要提高安全意识。

做技术的人，做一件事完全可以只需要几个关键词就能做好，因为很多前辈都已经铺好了路。从零开始对自己来说是好事。
